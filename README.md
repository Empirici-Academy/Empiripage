# Empirici Academy Website

This is the full-source code for Empirici Academy's website and admin panel for course and blog management.

## Features

- Fully responsive homepage, courses, about, blog, collaboration, contact & admin pages.
- Admin dashboard for CRUD operations on courses, blog posts, team, testimonials, stats, and social media/contacts.
- Data stored in localStorage and can be exported/imported as JSON.
- Homepage features selectable featured courses and blog posts.
- Horizontal scrolling for featured sections.
- Accessibility and SEO optimized.

## Setup & Usage

1. Clone or download the project files.
2. Serve on any static web server or open `index.html` directly.
3. Admin dashboard is at `admin.html` for managing content.
4. Edited data can be exported as JSON from admin and loaded via `data/site-data.json`.
5. Clear browser localStorage to reset data or reload from JSON file.

## File Structure

- `/assets` - Logo, favicon, and images.
- `/css` - Stylesheets.
- `/js` - JavaScript files.
- `/data` - JSON data file for initial content.
- Root: All HTML pages and admin dashboard.

## Recommendations

- Replace placeholder images in `data/site-data.json` with your own.
- Add favicon.ico derived from `assets/logo.svg` for better browser support.
- Use a local web server (like VSCode Live Server) for best development experience.
